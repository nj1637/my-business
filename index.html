Setup Analytics Tracking

To track real usage data, add this snippet to your main index.html inside the Firebase auth success block: 

// In your onAuthStateChanged callback:
const hitRef = doc(db, 'analytics', 'section_hits');
await updateDoc(hitRef, { [tabId]: increment(1) });

// Track invoice generation:
const invRef = doc(db, 'invoice_meta', invId);
await setDoc(invRef, {
  uid: user.uid,
  amount: grandTotal,
  date: new Date().toISOString(),
  client: clientName
}, { merge: true });
